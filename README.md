# O PROJEKCIE

Testowy projekt biblioteki zbudowanej w TypeScript.

Zawiera kilka przykładowych funkcji wraz z testami jednostkowymi napisanymi w Jest.

## Użycie

Uruchomienie testów

```
npm run jest
```

Build biblioteki

```
npm run build
```

Spakowanie biblioteki do paczki instalacyjnej

```
npm pack
```
